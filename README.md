### Project Start

#### Manual Start

+ yarn install //or use npm install
+ yarn serve
+ npx json-server --watch db.json 
+ node server/server.js

> Run in the project root (at 3 Terminals)

#### Quick Start
+ yarn install  //Install dependencies or use npm install
+ Then run start.bat from CMD
